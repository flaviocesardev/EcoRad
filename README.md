Uma Landing Page moderna, rápida e direta desenvolvida para a **EcoRad**, uma clínica especializada em diagnósticos e exames por imagem para animais.

🔗 **Acesse o site ao vivo:** [www.ecoradvet.com.br](https://www.ecoradvet.com.br)

---

## 🎯 Sobre o Projeto

O objetivo principal deste projeto foi criar uma vitrine digital profissional para a clínica. A página foi pensada para transmitir confiança aos tutores de pets e médicos parceiros, além de facilitar ao máximo o agendamento de exames e o contato com a equipe.

## ✨ Destaques da Solução

* **Design Responsivo:** O site se adapta perfeitamente a qualquer tamanho de tela (celulares, tablets e computadores).
* **Foco em Velocidade:** Desenvolvido com código limpo e leve para garantir um carregamento quase instantâneo.
* **Contato Facilitado (CTA):** Botões estratégicos posicionados pela página para direcionar o cliente rapidamente para o WhatsApp da clínica.
* **Presença Digital:** Estrutura otimizada para buscadores (SEO), ajudando a clínica a ter mais visibilidade no Google.
